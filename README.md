## Installation and Setup Instructions

Clone down this repository. You will need `node` and `yarn or npm` installed globally on your machine.

Installation:

`yarn or npm install`

To Run Test Suite:

`yarn or npm test`

To Start Server:

`yarn or npm start`

To Visit App:

`localhost:3000`
